# JustParkour

LÖVE2D로 만든 단일 파일 Parkour 횡스크롤 게임. 모든 로직은 [main.lua](main.lua)에 있습니다.

## 플레이어
- 스틱맨

## 액션
- 앞으로/뒤로 달리기
- 앞으로/뒤로 기어가기
- 위로 점프
- 엎드리기
- 앞으로 점프
- 뒤로 점프
- 앞으로 기어가기
- 뒤로 기어가기
- 기어오르기

## 실행
- LÖVE2D 11.x 설치 후 `run.bat` 실행 (또는 프로젝트 루트에서 `love .`).

## 조작
| 상황    | 키   | 동작           |
| ----- | --- | ------------ |
| 플레이 중 |     | 제자리 뛰기 애니메이션 |
| 플레이 중 | d   | 앞으로 달리기      |
| 플레이 중 | a   | 뒤로 달리기       |
| 플레이 중 | w   | 위로 점프        |
| 플레이 중 | s   | 엎드리기         |
| 플레이 중 | w+d | 앞으로 점프       |
| 플레이 중 | w+s | 뒤로 점프        |
| 플레이 중 | s+d | 앞으로 기어가기     |
| 플레이 중 | s+a | 뒤로 기어가기      |
## 플레이
- 게임을 로딩하면 플레이할 맵(도시)을 선택한다.
- 플레이어가 화면의 3/4으로 이동하면 우->좌 스크롤이 되면서 오른쪽에 새로운 장애물들이 나타난다.
- 플레이어가 화면의 1/4으로 이동하면 좌->우 스크롤이 되면서 왼쪽 이전 장애물들이 나타난다.
- 점프하여 장애물 옆 면을 잡으면 자동으로 장애물 윗 면까지 기어올라간다.
- 점프 시 효과음
- 목표물에 안착하면 맵 종료
