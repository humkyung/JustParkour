# JustParkour

LÖVE2D로 만든 단일 파일 Parkour 게임. 모든 로직은 [main.lua](main.lua)에 있습니다.

## 플레이어
- 스틱맨

## 액션
- 달리기
- 엎드리기
- 점프
- 기어오르기

## 실행
- LÖVE2D 11.x 설치 후 `run.bat` 실행 (또는 프로젝트 루트에서 `love .`).

## 조작
| 상황    | 키   | 동작         |
| ----- | --- | ---------- |
| 플레이 중 |     | idle 애니메이션 |
| 플레이 중 | d   | 달리기        |
| 플레이 중 | w   | 점프         |
| 플레이 중 | s   | 엎드리기       |
| 플레이 중 | w+d | 기어오르기      |
